# plantuml_to_fsm_tree_generator_example
В данном репозитории содержится пример использования module_fsm и plantuml_to_fsm_tree_generator.

# Сборка проекта
---------------------
```bash
git clone git@github.com:Vadimatorik/plantuml_to_fsm_tree_generator_example.git
cd plantuml_to_fsm_tree_generator_example
git submodule update --init --recursive
make pfsm_build
make all
```
